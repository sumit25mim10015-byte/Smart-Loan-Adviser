## Smart Loan Advisor – EMI & Affordability Calculator
# Overview of the project
This project is a simple Python-based Smart Loan Advisor that helps users calculate their monthly EMI and understand the affordability of a loan based on their monthly income.
It provides clear insights into:
•  Monthly EMI
•  Total payable amount
•  Total interest
•  Whether the EMI is affordable or too high
•  A warning message if the EMI exceeds safe limits
This makes it useful for anyone planning to take a personal loan, home loan, or education loan.


# Project Objective
The main aim of this project is to:
•	Calculate EMI using the standard loan formula
•	Allow users to enter loan amount, interest rate, and tenure
•	Provide advice based on the user’s monthly income


# Features of this project
•	Calculates EMI based on principal, interest rate, and tenure
•	Computes total payment & total interest
•	 Checks if EMI is affordable based on user’s income
•	 Gives warnings if EMI > 40% of income
•	Simple command-line interface


# How To Run This Project
  1.Install Python
   •  Python is installed on your computer.
  2. Clone the Repository

Clone using:
git clone https://github.com/your-username/Smart-Loan-Adviser.git

3. Run the Python Script
python loan_adviser.py

5. Enter the details when asked:
•     Loan amount
•     Annual interest rate
•     Loan tenure (in years)
•     Monthly income
The program will calculate everything and display the final Result.
